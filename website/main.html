Mapping the Galaxy

Network & Text Analysis in the Star Wars Universe

Introduction: A Data Analysis in a Galaxy Far, Far Away...

In this project, we analyze communication and conflict across the Star Wars universe using network and text analysis. The datasets includes characters, character attributes, and movie dialouges from Canon stories. Canon stories consists of movies, tv-shows, video games, and some novels. The data is gathered by web-scraping characters and character pages from Wookieepedia, and using StarWars API to fill out missing information. The movie
