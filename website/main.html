Mapping the Galaxy

Network & Text Analysis in the Star Wars Universe

Introduction: A Data Analysis in a Galaxy Far, Far Away...

In this project, we analyze communication and conflict across the Star Wars universe using network and text analysis. The dataset includes character dialogues, scene co-occurrences, and narrative arcs from the movies and extended universe.

  • Source: Star Wars scripts + fan wiki data
  • Size: 5000+ scenes, 100+ characters
  • Time Period: Episodes I–IX
