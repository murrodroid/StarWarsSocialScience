Mapping the Galaxy

Network & Text Analysis in the Star Wars Universe

Introduction: A Data Analysis in a Galaxy Far, Far Away…

This project presents a social-science analysis of the Star Wars Universe covering Episodes I-VI.

Using a dialogue dataset [3], we built a character interaction network: each character (node) is linked to every other character appearing in the same scene. By web-scraping Wookieepedia [1] and enriching it with the Star Wars API (SWAPI) [2], we added extensive metadata to each node, greatly enhancing our analysis.

Key Insights from the Galaxy

  • C-3PO is a network hub: highest in degree, betweenness, and closeness—bridging key storylines.
  • Communities mirror trilogies: clear separation between Prequel and Original Trilogy characters.
