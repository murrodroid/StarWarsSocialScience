Mapping the Galaxy

Network & Text Analysis in the Star Wars Universe

Introduction: A Data Analysis in a Galaxy Far, Far Away...

This project is a Social Science Data Analysis of the Star Wars Universe from I-VI episodes.

By using a dataset [3] containing all dialogues in the movie, we created a network.
Each living being (node) was connected (with edges) to everyone else featured within a given scene. By webscraping Wookieepedia [1] and applying Star Wars API (SWAPI) [2], we added massive amounts of metadata to each node which greatly supported our analysis.

Key Insights from the Galaxy

  • C-3PO is a network hub: Highest ranked in degree, centrality, and closeness. Acts as the bridge between storylines
  • Communities reflected Trilogies: A clear seperation between characters from the two trilogies appeared
